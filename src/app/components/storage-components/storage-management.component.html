Storage management

This component is for the management of the data stored within this browser's database.

Export browser database

The following button will export the browser database to a .json file.

Empty current database

The following button will wipe your browser database. This is not reversable. The browser database will need to be restored from a prior .json file.

Append to database from file

The following button will append the contents of the given .json file to your browser database. If a key within your .json file matches that of your browser database then the database entry will be overwritten. This is not reversable.
