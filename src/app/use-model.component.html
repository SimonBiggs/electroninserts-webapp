Use models

Component still under construction

This component will be where the created models will be used. Linear interpolation will be used over the 0.1 cm resolution grid returned by the model creation. Factors for prediction will be able to be inputted using width, and then either length or area, and optionally a measured factor can be included for the purpose of comparing model results with measurement.

You can either use the parameterisation component to parameterise your shapes, or if you have a large data set of generic shapes you need only determine the diameter of the largest encompassed circle to define width, and then you may input shape area instead of length. This allows you to quickly check large data sets of previous measurements on generic shapes.
